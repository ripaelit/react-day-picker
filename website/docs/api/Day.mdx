<a id="docs"></a> 

The `Day` component renders the content of the day cell. It renders a button
if the day is interactive (i.e. it is clickable).



<a id="props"></a>

# Day props


## currentMonth
The month the calendar belongs to. Used to calulate outside days.
<a class="syntax"></a>

`prop `**`currentMonth:`** [`Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date)

<hr />




## day
The day to display in the calendar.
<a class="syntax"></a>

`prop `**`day:`** [`Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date)

<hr />




## dayPickerProps
Reference to the props used by the DayPicker component.
<a class="syntax"></a>

`prop `**`dayPickerProps:`** [`DayPickerProps`](./interface#daypickerprops)

<hr />
