<a id="docs"></a> 

Hook to bind a input with a calendar.

```jsx
const {
 month,
 selected,
 dayPickerProps,
 inputProps
} = useInput(new Date());

<DayPicker {...dayPickerProps} />
<input {...inputProps} />
```
