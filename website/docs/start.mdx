> DayPicker helps React developers to implement date pickers without worrying about JavaScript dates and layouts.

:::important

This documentation is about the upcoming version, which is still in development.
See [react-day-picker.js.org](http://react-day-picker.js.org) for the
documentation of the current version.

:::

## Installing and importing

Add `react-day-picker` to your projectâ€™s dependencies:

```bash
yarn add react-day-picker@next     # when using yarn
npm install react-day-picker@next  # when using npm
```

Import DayPicker and its CSS file in your script.

```jsx
import "react-day-picker/dist/style.css";
import { DayPicker } from "react-day-picker";
```
