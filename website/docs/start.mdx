#### Installing DayPicker

1. Add [`date-fns`](https://date-fns.org/) and `react-day-picker` to the projectâ€™s dependencies:

```bash
# when using yarn
yarn add date-fns react-day-picker

# when using npm
npm install date-fns react-day-picker
```

2. import DayPicker in your script.

```jsx preview
import { DayPicker } from 'react-day-picker';

function App() {
  return <DayPicker onMonthChange={console.log} onDayClick={console.log} />;
}
```
