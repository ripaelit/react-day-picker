---

Use the [useInput hook](/api/functions/useinput) to bind an input field with
DayPicker.

1. call `useInput` hook by passing the initial day and the [format
   string](https://date-fns.org/v2.17.0/docs/format);
2. use the returned props for the `DayPicker` and the `input` component;
