> Changing the content of the DayPicker components it is easy with [formatting
> props](./reference/props.mdx#formatting).

Use [formatting props](./reference/props.mdx#formatting) to format the content of the day
cell, the month’s caption, the weekday name or the week’s number.

* For more advanced customization of those elements, see [custom components](./custom-components.mdx).
* Formatting props can be used for [localization](./localization.mdx).

```jsx preview
import React, { useState } from 'react';
import { DayPicker } from 'react-day-picker';
import { format } from 'date-fns';

function App() {
  const formatDay = day => {
    const suffix = day.getDay() === 2 ? '*' : '';
    return `${day.getDate()}${suffix}`;
  };

  const formatCaption = (month, { locale }) => {
    return `${format(month, 'dd/yyyy', { locale })}`;
  };

  const formatWeekdayName = (day, { locale }) => {
    return format(day, 'E', { locale })[0];
  };

  return (
    <DayPicker
      showOutsideDays
      showWeekNumber
      formatDay={formatDay}
      formatCaption={formatCaption}
      formatWeekdayName={formatWeekdayName}
    />
  );
}
```
