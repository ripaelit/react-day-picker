## Style via CSS

DayPicker includes a [CSS
file](https://github.com/gpbl/react-day-picker/blob/master/packages/react-day-picker/style.css)
to import or to use as template.

```js
import 'react-day-picker/style.css';
```

The CSS uses few variables to quickly override the colors and the cell size:

```css
:root {
  --rdp-font-family: sans-serif;
  --rdp-accent-color: #4d21fc;

  /* Default colors for light themes */
  --rdp-accent-color-light: #8567fd;
  --rdp-accent-color-lighter: #beaefe;
  --rdp-accent-color-lightest: #f6f4ff;

  /* Switch to dark colors for dark themes */
  --rdp-accent-color-dark: #3003e1;
  --rdp-accent-color-darker: #1e0097;
  --rdp-accent-color-darkest: #180270;
  --rdp-cell-size: 40px;
  --rdp-border-focus: 2px solid var(--rdp-accent-color);
}
```

## Custom CSS classes

Replacing the default class names is useful with [CSS
Modules](https://github.com/css-modules/css-modules) or other JS-in-CSS
bundlers.

To use custom classes, use `classNames`. [`UIElement`](/api/enums/uielement)
contains a list of the elements that is possible to style.
