title: Changing Months
---

As default, the current month is stored into the DayPicker’s internal state. To
change the month programmatically, the current month must be stored in the
parent's state.

1. save the month to render in the parent component’s state
2. use the `month` prop (as opposed to `defaultMonth`)
3. use the `onMonthChange` event handler to set the month

```jsx showOutput
function Example() {
  const [selected, setSelected] = useState();
  const [month, setMonth] = useState();

  const handleDayClick = (day, { selected }) =>
    setSelected(selected ? undefined : day);

  return (
    <>
      <DayPicker
        month={month}
        selected={selected}
        onMonthChange={setMonth}
        onDayClick={handleDayClick}
      />
      <button onClick={() => setMonth(new Date())}>Go to Today</button>
    </>
  );
}
```
