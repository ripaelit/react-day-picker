### Controlling the current month

The current month is stored into the DayPicker’s internal state.

To control the current month – for example, to create a "Go to today"
button – set it in the parent’s state.

Use `month` (as opposed to `defaultMonth`) and `onMonthChange` to handle the
current month.

import Uncontrolled from '../examples/navigation-uncontrolled.tsx';
import UncontrolledSrc from '!!raw-loader!../examples/navigation-uncontrolled.tsx';

## Limiting the navigable months

As default, DayPicker will display indefinitely in the past or in the future.

To limit the navigable months between certain dates, use `fromDate` and `toDate`, `fromMonth`
and `toMonth`, or `fromYear` and `toYear`.

import FromTo from '../examples/navigation-from-to.tsx';
import FromToSrc from '!!raw-loader!../examples/navigation-from-to.tsx';
