## Change the initial month

As default, DayPicker displays the month of the current day. To change the
initial month, set the `initialMonth` prop:

```jsx showOutput open=no
function Example() {
  return <DayPicker initialMonth={new Date(2022, 8)} />;
}
```

## Month navigation

As default, DayPicker navigates indefinitely in the past or in the future.

To limit the months to certain dates, use `fromDate` and `toDate`, `fromMonth`
and `toMonth`, or `fromYear` and `toYear`.

```jsx showOutput open=no
function Example() {
  const fromDate = new Date(2018, 6, 15);
  const toDate = new Date(2020, 10, 15);
  const fromMonth = new Date(2020, 4);
  const toMonth = new Date(2022, 9);
  const fromYear = 2010;
  const toYear = 2015;
  return (
    <>
      <DayPicker initialMonth={fromDate} fromDate={fromDate} toDate={toDate} />
      <div>
        From {fromDate.toLocaleDateString()} to {toDate.toLocaleDateString()}
      </div>
      <DayPicker
        initialMonth={fromMonth}
        fromMonth={fromMonth}
        toMonth={toMonth}
      />
      <div>
        From {fromMonth.toLocaleDateString()} to {toMonth.toLocaleDateString()}
      </div>
      <DayPicker
        initialMonth={new Date(fromYear, 0)}
        fromYear={fromYear}
        toYear={toYear}
      />
      <div>
        From {fromYear} to {toYear}
      </div>
    </>
  );
}
```

### Using a dropdown to navigate between months

When limiting the navigable months, set `captionLayout` to display a drop-down
for navigating between months.

```jsx showOutput open=no
function Example() {
  return <DayPicker fromYear={2015} toYear={2025} captionLayout="dropdown" />;
}
```

### Disabling navigation

To disable navigation between months, set `captionLayout` to `none`:

```jsx showOutput open=no
function Example() {
  return <DayPicker fromYear={2015} toYear={2025} captionLayout="none" />;
}
```

## Showing the outside days

Use `showOutsideDays` to display the days falling out the current month.

```jsx showOutput open=no
function Example() {
  return <DayPicker showOutsideDays />;
}
```

## Using fixed weeks

When `showOutsideDays` is turned on, use `fixedWeeks` to display six weeks per
months. This will prevent the calendar to change its height when navigating.

```jsx showOutput open=no
function Example() {
  return <DayPicker showOutsideDays fixedWeeks />;
}
```

## Showing the week numbers

Use `showWeekNumber` to display the week numbers.

```jsx showOutput open=no
function Example() {
  return <DayPicker showWeekNumber />;
}
```

## Multiple months

Use the `numberOfMonths` to render more than one calendar.

When rendering multiple months, use the following props:

- `pagedNavigation` – to navigate the number of months per time;
- `reverseMonths` – to display the latest month first.

```jsx showOutput open=no
import { DayPicker } from 'react-day-picker';

function Example() {
  return <DayPicker numberOfMonths={2} reverseMonths={false} pagedNavigation />;
}
```
